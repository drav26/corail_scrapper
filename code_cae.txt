
	  	
	
Code d'activité économique (CAE)
	
	
 
	
	

Un code d'activité économique (CAE) est un code à quatre chiffres qui désigne un secteur d'activité.

Appuyez simultanément sur les touches « CTRL » et « F » pour effectuer une recherche par mot-clé.

Note

* Ces codes d’activité économique ne peuvent être utilisés que par les professionnels membres de l’ordre professionnel visé.

** Ce code d’activité économique ne peut être utilisé que par les titulaires d’un permis de courtage immobilier valide exigé pour la pratique de telles activités et délivré par l'Organisme d'autoréglementation du courtage immobilier du Québec.

*** Ce code d’activité économique ne peut être utilisé que par les titulaires d’un certificat de courtier hypothécaire valide délivré par l’Autorité des marchés financiers conformément à la Loi sur la distribution de produits et de services financiers.
